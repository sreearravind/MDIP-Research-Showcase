# MDIP Technical Scope

## Public showcase scope

The public showcase scope includes project-level documentation, architecture descriptions, methods summaries, implementation maturity overviews, proposal alignment notes, and figure placeholders intended for reviewer communication. This scope is designed to explain what MDIP is, why it matters, how it is structured, and what has been implemented at a high level.

## Private implementation scope

The private implementation scope includes backend source code, service-layer logic, API implementation files, access control enforcement details, import and workflow internals, report generation internals, deployment-sensitive assets, environment configuration, migrations, local storage behavior, and any partner- or institution-linked operational materials. These items remain outside the public repository and may be shared only through controlled technical review processes.

## Future research scope

The future research scope includes analytics/SPC development, Weibull reliability analytics, predictive machine learning models, RAG/SLM-supported engineering intelligence, cloud-scale orchestration, async/background workflow execution, and broader user-facing interface development. These items represent the forward research and scaling roadmap that builds on the current backend foundation.
