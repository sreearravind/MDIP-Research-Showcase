# Access Policy

## Controlled Access Position

The full MDIP backend implementation is maintained in a private repository and is not distributed through this public showcase repository.

## Eligible Access Contexts

Access to the private implementation may be granted only for the following purposes:

- proposal review
- technical evaluation
- research collaboration
- institutional review

## Public Repository Exclusions

This showcase repository intentionally excludes:

- source code for backend services and internal APIs
- credentials, tokens, and environment configuration
- private datasets and partner-specific records
- deployment secrets and infrastructure-sensitive files
- implementation-sensitive files that could expose internal operational details

## Access Conditions for Private Review

Where private access is approved, it should be provided as read-only unless otherwise approved in writing by the project owners or relevant institution.

## Redistribution Restriction

Reviewers, evaluators, and collaborators should not redistribute private code, internal documentation, or related implementation materials without written permission.

## Data Protection Intent

This policy is designed to support transparent proposal review while protecting confidential implementation logic, sensitive operational details, and any data connected to institutional or MSME stakeholders.
